# San Francisco Crime Analysis (2003–2025)

## Overview
This project analyzes how the composition of crime in San Francisco changed before, during, and after COVID-19.  
The focus is on crime composition (percentages) across districts rather than total counts.

---

## Live Project
https://rupakkadhare15.github.io/02806-Social-Data-Analysis-and-Visualization/

---

## Visualizations
- Stacked bar chart: Crime composition over time (2003–2025)  
- Map: District-level comparison (pre vs post COVID)  
- Line chart: Pre, during, and post COVID comparison across districts  

---

## Data Source
San Francisco Police Department Incident Reports (SF Open Data)
